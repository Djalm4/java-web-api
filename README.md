My First Web API – Spring Boot

Este projeto é uma API REST desenvolvida em Spring Boot com foco no aprendizado dos conceitos básicos de desenvolvimento backend.
A aplicação simula o gerenciamento de usuários, contendo operações como criação, listagem, busca, atualização e exclusão, utilizando uma camada de repositório mockada (sem banco de dados).

Tecnologias:

Java 21

Spring Boot

Spring Web MVC

Maven

Objetivo:

Praticar APIs REST

Entender arquitetura em camadas
